What is this?

The Black–Scholes model ↗ is a mathematical framework for pricing European-style options contracts. Given five inputs — stock price, strike price, time to expiry, risk-free rate, and implied volatility — it produces a fair-value estimate for a call or put, and a set of sensitivities known as the Greeks ↗.

This tool lets you drag those inputs in real time and watch how Delta ↗, Gamma ↗, Theta ↗, and Vega ↗ shift — built from first principles using the closed-form Black–Scholes equations, no libraries.
